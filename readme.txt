Olá, este é o teste para a vaga de desenvolvedor Front-End da hubert, 
esse teste visa avaliar suas competências em Javascript, html e css, para assim
 averiguar se está ou não apto a exercer a vaga em questão.



Na pasta Layout, você encontra .png com a tela que você deve realizar o front.
Torne todos os elementos desta página funcionais, faça a validação dos campos,

- ignorar os itens: Agendamentos, Inventário, Vistoria, Relatórios e Regiões & Locais.(Se desejar pode deixar eles linkando para um página em branco)
- o campo telefone precisa usar a máscara apropriada
- o campo Qtd de cameras precisa aceitar apenas números
- o campo Tipo tem as opções Apartamentos/Casas
- o campo Proteção perimetral tem as opções Não verificado/ Verificado
- o campo Antenas tem as opções Não possui / Possui
 
E de preferência, deve exibir dados de 2 condomínios, lendo e salvando os dados em um arquivo Json.
Se atentar aos campos que podem ou não ser editáveis.
O seletor de condomínio pode estar na própria página ou na barra superior, o que for mais fácil. 

Tecnologias obrigatórias:
HTML5
CSS3
JavaScript
bootstrap

DIFERENCIAL (Não Obrigatório):
Você pode optar e fazer esta tela usando react ou angular, mas ao utilizar envie o projeto com uma documentação de rodar sua aplicação.

Lembre-se, organização, limpeza do código e boas práticas,tecnologias atuais são importantes e serão avaliadas.
